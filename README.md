# Gensokyo-NewQQ

面向 QQ 官方机器人 API 的 OneBot V11 网关与兼容层项目。

本仓库用于维护 Gensokyo-NewQQ 的核心实现。项目目标是在 QQ 官方机器人能力与 OneBot V11 生态之间提供一层较稳定的协议转换、事件整理和接口适配，方便既有机器人框架、插件和管理工具接入。

## 项目定位

`Gensokyo-NewQQ` 更接近网关与协议桥接组件，而不是单个机器人插件。它主要负责连接 QQ 官方机器人接口、内部事件整理逻辑和 OneBot V11 风格的外部调用方式。

可能覆盖的能力包括：

- QQ 官方机器人事件接入
- OneBot V11 API 兼容
- WebSocket / HTTP 通信
- 群消息与私聊消息处理
- 扩展 CQ 码或扩展 API 支持
- WebUI 或管理入口配套能力

具体支持范围请以代码和配套文档站为准。
